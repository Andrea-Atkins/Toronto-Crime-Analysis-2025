# Toronto-Crime-Analysis-2025
End-to-end data analytics project using Excel, SQL and Power BI to analyze crime patterns across Toronto neighbourhoods in 2025.
## Project Overview

This project analyzes non-lethal crime incidents reported across Toronto during 2025 using Excel, SQL and Power BI.

The objective was to identify crime patterns by neighbourhood, offence type, and time period while providing data-driven recommendations to improve public safety strategies.
